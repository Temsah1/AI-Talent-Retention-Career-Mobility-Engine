# 🚀 AI Talent Retention & Career Mobility Engine

An Enterprise-grade Machine Learning & NLP application built to predict employee churn risk and recommend intelligent internal career paths, helping HR teams make data-driven decisions.

## ✨ Features
* **🔐 Secure Authentication:** Built-in Role-Based Access Control (Admin vs. User).
* **📊 HR Executive Dashboard:** Real-time analytics, retention metrics, and correlation heatmaps using Plotly.
* **⚠️ Flight Risk Predictor:** Random Forest Machine Learning model predicting employee churn probabilities with explainable AI (Feature Importance).
* **🧠 Smart Career Mobility:** NLP-powered recommendation engine using TF-IDF and Cosine Similarity to suggest internal roles and identify skill gaps.

## 🛠️ Tech Stack
* **Frontend/Backend:** Streamlit (Python)
* **Data Processing:** Pandas, NumPy
* **Machine Learning:** Scikit-Learn (RandomForest, TF-IDF Vectorizer)
* **Data Visualization:** Plotly Express & Graph Objects
* **Database:** SQLite3
